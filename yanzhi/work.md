## 页面详解
### WELCOME
| widget(页面组件) | 解释                                                                    |
| ---------------- | ----------------------------------------------------------------------- |
| 颜值日记宣传图   | 第一个宣传页面,介绍跟颜值日记相关的元素,比如点赞,金币,热度等等          |
| 周冠军宣传图     | 第二个宣传页面,介绍跟周冠军相关的元素,比如周冠军的礼品,活动每周都会比赛 |
| 种植宣传图       | 第三个宣传页面,介绍女生特有的种植元素,比如种子,收获                     |
| 注册用户基本信息 | 进入app之前必须填写的基本信息,包括昵称,城市,生年和性别                  |
 
### HEADER

 | widget(页面组件) | 解释                                                        |
 | ---------------- | ----------------------------------------------------------- |
 | 快速签到         | 如果当天还没有签到,点击就可以快速签到;如果已经签到了,就隐藏 |
 | 快速转发         | 如果当天还没有转发,点击就可以快速转发;如果已经转发了,就隐藏 |
 | 当前金币         | 显示用户当前的金币                                          |
 | 站内信按钮       | 快速跳转到站内信页面;如果有未读的站内信,需要红点标记        |

### 我的信息
| widget(页面组件) | 解释                                                                       |
| ---------------- | -------------------------------------------------------------------------- |
| 我的基本信息     |
| 签到日历         | 以日历形式展示当月的签到情况;用户可以在这里签到;显示`未签` `签到` `大签到` |
| 金币流水页面     | `简短的文本`,`时间`和`金币的流向`来向用户展示每一笔流水                    |
| 勋章             | 显示我所有得到的勋章;有的勋章是可以计数的                                  |
 ** 周六,周日是大签到,可以获得比平日额外的金币奖励
### 交友界面
  | widget(页面组件) | 解释                                                                             |
  | ---------------- | -------------------------------------------------------------------------------- |
  | 筛选器           | 是否只筛选同一个城市的女生;如果男生没有选择城市,需要弹框提醒他去设置他所在的城市 |
  | 女生形象照       | 形象照,类似探探,左滑右滑(左滑表示略过,右滑表示**查看详情**)                      |
  | 勋章             | 展示3个权重最大的勋章;官方认证是最大的权重                                       |
  | 年龄             |                                                                                  |
  | 热度             | 超过1000,用单位k来友好显示;比如女生热度为12450,则显示为12k                       |
  | 最后一次登录时间 | 超过1天,用1D;超过1个月,用1M;超过1年,显示为失踪人口;否则显示为最近                |
	

## 颜值日记
| widget(页面组件) | 解释                                                                      |
| ---------------- | ------------------------------------------------------------------------- |
| 女生形象照       | 位置在最顶部,显示女生的基本信息和访问者的热度,                            |
| 关注按钮         | 如果访问者没有关注者需要显示`关注`;已经关注的,可以反关注,但是要提醒该操作 |
| 赠送礼物按钮     | 弹出礼物选择器                                                            |
| 聊天按钮         | 进入聊天页面                                                              |
| 媒体列表         | 媒体类型;点赞按钮,点赞只能一次;被女生设置为出售的媒体,需要特殊标记        |
| 单个日记详情     | 图片弹出高清图片;视频和音频具有播放按钮                                   |

**媒体类型有三个种类,图片,短视频和语音**
**关注需要金币,如果男生没有金币,需要给他一个友好提示框**

## 聊天
| widget(页面组件) | 解释                                                   |
| ---------------- | ------------------------------------------------------ |
| 聊天             | 一般聊天功能                                           |
| 附加金币         | 男生可以在每条聊天信息中附加金币;女生回复了就可以领取; |
| 赠送礼物         | 点击弹出礼物商店                                       |
| 赠送种子         | 点击弹出种子商店                                       |
| 发送语音         |                                                        |
| 线下邀请         | 待定                                                   |
**3天不回复,金币会以站内信方式返回给男生**
**热度不够的功能需要灰色处理**

## 颜值日记上传页面   
| widget(页面组件) | 解释                                              |
| ---------------- | ------------------------------------------------- |
| 媒体上传         | 标题+媒体                                         |
| 设置出售         | 女生可以把自己的个别媒体设置为出售,比如"叫醒语音" |
| 设置隐藏         | 暂时不向他人开放                                  |

## 种植页面           
| widget(页面组件) | 解释                                                                                     |
| ---------------- | ---------------------------------------------------------------------------------------- |
| 农场             | 种子的成长情况;铲子(用以种植);洒水壶(用以浇水);化肥包(用以施肥);镰刀(种子成熟,用以收割); |
| 种子商店         | 我的种子;在售的种子;在售的化肥                                                           |


**每个女生一个时间里只能种植一个种子**
**种子需要有4个不同的图来表示不同的成长状态**
**男生可以帮忙浇水,消耗金币**

## 站内信             
| widget(页面组件) | 解释                                     |
| ---------------- | ---------------------------------------- |
| 未读删选器       | 勾选之后只暂时未读的信件                 |
| 信件列表         | 缩略的文本;发信日期;发信人;附件类型;     |
| 信件详情         | 完整的文本;发信日期;发信人;附件;领取按钮 |

**附件可能为金币,勋章,种子等等**
**有附件的信件需要领取才能被确认为已读**
